About Seedr

Seedr provides easy methods to download and transfer files directly without using any external vpn or anything else.

With Seedr, you can download anything when you want it, without the hassle of having to download, unzip, and keep track of multiple files.
Using seedr is very easy. Here is a brief description how to get onboard with seedr



Getting started


  • Go to the site https://www.seedr.cc/
  • Create a personal account using eamil or facebook account
  • After creating account verify your email/facebook address
  • After signup is completed, login to seedr.cc
  • After logging into seedr you can see a textbox on the top right corner, where you can paste the link you want to download.
  • After pasting the address, the list of files apprears below and you can download your file



Seedr benefits



1. By default seedr provides 2gb free memory to every user. Any file within size 2gb can be downloaded absolutely free.
2. There is also an option to opt for a premium paid service.
3. You can also increase the storage by refering to 4 friends and earn upto 2gb extra.
4. Post on twitter or personal blog to get further storage extension.
